== Description ==
"Windy Citizen Share" is a plugin that allows you to place <a href="http://www.windycitizen.com">The Windy Citizen's</a> (a social news site for Chicago/Illinois news) <a href="http://windycitizen.com/buttons">"Share" button</a> at the end of your posts, pages, or anywhere at your choice. While it will run out of the box, it has several options that can be used to custom the button to fit your needs:

* Automatically add "Windy Citizen Share" button to your posts (yes/no). Default is set to Yes.
* Automatically add "Windy Citizen Share" button to your pages (yes/no). Default is set to Yes.
* Background color *around* the button. The option is provided with a handy color picker. Default is "#ffffff" (white)
* Skin of the button. The Windy Citizen offers 3 skins: Tall, Wide and Text. Default is set to "Tall" ;)

== Installation ==
1. Download and extract the plugin
1. Upload the entire `windy_citizen_share` folder into to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. By default your posts and pages should now have a "Windy Citizen Share" button at the end. If you want more power:
1. Head to Settings -> Windy Citizen Share and configure the options there

== Frequently Asked Questions ==
= I want to customize the margin/padding/style of the div that wraps the button. How? =
The button is wrapped in a div with "dtom" class. So just add .dtom into your style.css and modify it.
